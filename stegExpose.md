** Using StegExpose to detect hidden data in images** 
step1:download StegExpose from github 
step2:download jdk for windows
step3:extract the StegExpose
step4:open stegExpose with terminal
step5:use the command "java -jar StegExpose.jar "folder_path"
step6:if score is less than 0.2 then no hidden data
step7:if score is between 0.2-0.3 then hidden data exist 
step8:if score is greater than 0.3 then stegnography
